# NumSysAsm
A simple calculator for working with various number systems (2 &lt;= x &lt;= 16) written in NASM assembly
## Использование
Данную программу можно использовать для перевода целых положительных чисел из (2 <= n <= 16) систем счисления в десятичную систему счисления.
К сожалению, ввиду некоторых сложностей с использованием некоторых функций WinApi до конца не был реализован механизм ввода с буфером и динамическим размером через HeapAlloc. Поэтому для "ввода" необходимо вручную изменять код самой программы. Благо, компилируется всё очень быстро.
Входное число записывается в "переменную" origNumber (на самом деле это указатель), а основание системы счисления во вторую операнду инструкции mov в 31 строке файла.
Также реализован механизм обработки ошибок в рамках некорректных символов в вводе.
Снизу прикреплено видео с демонстрацией работы программы.
